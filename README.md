# Kurisho

## Getting started

    # Clone repository
    $ git clone https://github.com/JacobC1921w/Kurisho.git
    $ cd Kurisho

    # Update submodules
    $ git submodule init
    $ git submodule update

    # Install pip requirements
    python3 -m pip install -r requirements.txt

    # Run Kurisho
    ./index.py
